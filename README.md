# 项目简介

## 核心功能
- 基于 YOLOv8 的车辆朝向检测系统，支持 ['back', 'front'] 两类车辆朝向识别
- 提供完整源码、3500 张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能包括：实时显示目标位置、目标总数、置信度信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供批量检测功能，可查看所有检测结果
- 支持检测结果保存功能
- 提供完整项目运行教程

## 适用场景
- 车辆管理系统的智能识别模块
- 交通流量监测与分析
- 停车场车辆朝向统计
- 车辆行为研究
- 智能监控系统

## 技术特点
- 使用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 基于 YOLOv8 模型进行车辆朝向检测
- 提供图形化用户界面，操作简便
- 配置完整环境后可直接使用
- 支持多种检测模式和参数调整
